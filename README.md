# MegaToolSuite

MegaToolSuite is a modular orchestration framework for automated pipelines and jobs.

## Features
- YAML‑based pipeline definitions
- Built‑in plugins (file operations, HTTP, data transformations, notifications)
- Multiple execution runners (local, threaded, async)
- Configurable profiles (dev, prod, test)
- Complete example pipelines included

## Installation
Use the PowerShell installer:

```powershell
.\Install-MegaToolSuite.ps1
```
